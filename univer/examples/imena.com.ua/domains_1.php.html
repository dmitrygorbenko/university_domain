  • Контакты
    Здесь Вы можете сменить контакты для данного доменного имени:
    • owner-c Владелец доменного имени. Сертификат на данное имя выдается на Компанию, так как она указана в поле Company, или, если это поле не Заполнено, на частное лицо, как указано в полях Firstname и Lastname. Подтверждение от owner-c нужно при переводе доменного имени к другому регистратору. Чтобы изменить контаст нажмите edit contacts.
    • admin-c Администратор доменного имени, лицо которое занимается администрированием Доменного имени и уполномочено производить любые изменения.
    • billing-c Контакт на который будут приходить уведомления по оплате доменного имени,
      Счета и т. д.

  • Настройка домена
    Здесь можно внести изменения в файл зоны на Name Servers Предоставляемых интернет-службой IMENA.com.ua Можно прописать IP или CNAME для домена и субдоменов, А также MX записи для почты.

  • Web редирект
    Здесь можно установить переадресацию WEB Укажите в поле Redirect to URL: (лучше скопировать URL используя комбинации ctrl+c, ctrl+v ) адрес на который будет переадресовываться данный домен и нажмите Update. Если Вы не желаете, видеть в браузере реальный URL выберете вместо direct With frames и нажмите Update. В поле Title: Вы можете ввести надпись, которая будет выводиться в заголовке браузера. Чтобы убрать переадресацию, отметьте clear redirect и нажмите Update. Переадресация работает, если все записи в меню [zone] сделаны по умолчанию.

  • Почтовый редирект
    Здесь можно установить переадресацию почты
    Укажите в поле Redirect to: e-mail на который будет приходить вся почта, направляемая на данное доменное имя и нажмите кнопку Update.
    Чтобы убрать переадресацию, отметьте clear redirect и нажмите кнопку Update.
    Переадресация работает, если все записи в меню [zone] сделаны по умолчанию.

  • NS сервера
    Здесь можно изменить Name Servers для данного доменного имени.

  • Заказ сертификата
    Здесь Вы можете заказать сертификат для данного доменного имени.




домены
Здесь можно просмотреть все доменные имена, которые Вы регистрировали, получить их сервис-коды и даты окончания регистрации. Также здесь можно изменить контакты для домена, прописать IP адрес и MX записи, настроить WEB и MAIL переадресацию, сменить Name Servers.
 
профайл
Здесь хранится информация, определяющая учетную запись пользователя, регистрирующего все доменные имена, ассоциируемые с данным аккаунтом. Этот контакт может не относится ни к одному домену, он относится к учетной записи, с которой связаны логин и пароль, а также сумма на балансе.
 
пароли
Здесь можно создать отдельный аккаунт, для управления всеми Вашими Доменами или некоторыми из них. Также можно отключить аккаунт ставший ненужным.
